Bpress - Blog para criação de artigos

O projeto é basicamente um site em formato de blog onde os usuários podem criar e acessar vários artigos que possuem suas categorias, o blog possui um sistema de login para administrador onde o adm tem controle total.Foi feito com o padrão MVC (Model-View-Controller), utilizamos um Middleware para fazer a autenticação do o usuário esta logado, tinymce para poder deixar a area de criação de artigo mais dinamica.
